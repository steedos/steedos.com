## 安装前准备

- 安装[docker](https://docs.docker.com/get-docker/)
- 安装[docker-compose](https://docs.docker.com/compose/install/)
- 由于镜像存在国外，可配置阿里云[官方镜像加速](https://help.aliyun.com/document_detail/60750.html)

## 私有部署

访问 [Steedos Docker](https://github.com/steedos/docker) 项目仓库， 下载 docker-compose.yml 模版文件。

- [Github下载](https://github.com/steedos/docker/blob/main/steedos-community/docker-compose.yml)
- [Gitlab下载](https://gitlab.steedos.cn/steedos/docker/-/raw/main/steedos-community/docker-compose.yml)

### 选择 steedos 版本

可以参考[docker hub](https://hub.docker.com/r/steedos/steedos-community/tags?page=1&ordering=last_updated)选择合适的版本号，通常建议运行最新版的 steedos 。

### 设定环境变量

参考文档 [配置环境变量](./steedos-config)

```yaml
environment:
  - PORT=3000
  - ROOT_URL=#在此填入服务运行的正式网址，包含端口号3000。例如 http://192.168.0.200:3000
```

### 启动服务

使用 docker 启动服务

```yaml
docker-compose up -d
```

查看服务状态

```yaml
docker ps
```


停止服务

```yaml
docker-compose stop
```


停止并删除服务

```yaml
docker-compose down
```
